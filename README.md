# WinUtils

## 项目介绍

C++23 现代Windows微功能组件库，所有功能均同时支持宽窄字节

## 功能介绍

1. 轻量命令行解析器，支持多种解析方式
2. 控制台常用功能封装
3. 控制台菜单，支持多级菜单，命令调用
4. HTTP网络库，封装winsock，支持Get、Post方法
5. Dll注入器
6. Windows服务管理
7. 字符串转换
8. 进程/线程/窗口操作
9. 完善单文件日志库
10. 各类加密算法实现
11. INI文件解析
12. ...

## 环境依赖

Visual Studio 2022

## 项目依赖

[hashlib](https://github.com/stbrumme/hash-library)(C++加密库)

[mINI](https://github.com/metayeti/mINI)(C++INI解析器)
