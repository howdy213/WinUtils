# WinUtils

## Project Introduction

C++23 modern Windows micro-functionality component library, all features support both ANSI and Unicode character sets

## Features

1. Lightweight command-line parser supporting multiple parsing methods
2. Encapsulation of common console functions
3. Console menu with multi-level menu support and command invocation
4. HTTP network library, encapsulating Winsock, supporting GET and POST methods
5. DLL injector
6. Windows service management
7. String conversion
8. Process/thread/window operations
9. Comprehensive single-file logging library
10. Various encryption algorithm implementations
11. INI file parser
12. Registry reading and writing
13. ...

## Environment Dependencies

Visual Studio 2022
